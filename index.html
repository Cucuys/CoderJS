Entrega uno

Soy aficionado a la aviacion. La entrega trata de un simulador de rendimiento para el despegue de un avión, el simulador toma en cuenta peso, temperatura y longitud de la pista, los modelos seleccionadon son el A320, A350 Boing 737. Si bien se utilizan mas variables para el calculo, se obvian para un futuro y se toman peso, temperatura y altitud. Dejé comentarios explicando ya que me costó bastante usar funciones de flecha, las realicé "normales".Por ultimo, dirigo el proyecto para al finar realizar un simulador de despegues con mas variables.
